# Python Learning Project

Educational project for studying Python, Streamlit and API integration.

## About

This repository contains various Python experiments and learning materials:
- Streamlit dashboard examples
- API integration patterns
- Data analysis scripts
- Testing examples

## Structure

```
py_study/
├── dashboard.py          # Main Streamlit application
├── api_client.py         # API interaction examples
├── data_manager.py       # Data processing
├── tests/               # Unit tests
└── docs/                # Documentation
```

## Setup

```bash
pip install -r requirements.txt
streamlit run dashboard.py
```
